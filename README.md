# 🔐 ** Python Password Manager (Tkinter)**
### A desktop Password Manager built in Python using Tkinter, designed to demonstrate clean code structure, user-focused features, and real-world data handling.

> This project goes beyond basics by combining GUI development, file persistence, validation logic, and automation into one cohesive application.

## 🚀 Features

- 🔐 Secure password generator (letters, numbers, symbols)

- 📋 One-click clipboard copy

- 💾 Persistent storage using JSON

-  🔍 Search saved credentials by website

- ✉️ Auto-fill most frequently used email

- 🧠 Input validation and error handling

- 🖥️ Clean and responsive Tkinter GUI

## 🗂️ Project Structure

- main.py — Application logic and GUI

- data.json — Stored credentials (ignored in version control)

- logo.png — Application logo

- .gitignore — Prevents sensitive/local files from being committed

## 🧪 How It Works

1. Enter a website and email

2. Generate a secure password or enter your own

3. Confirm and save credentials

4. Data is stored locally in a structured JSON file

5. Search any saved website to instantly retrieve credentials
